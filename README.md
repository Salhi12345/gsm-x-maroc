## 📌 About The Project

GSM X Maroc is a static website designed to showcase GSM services in Morocco including:

- 📱 Mobile unlocking services  
- 💻 Software support tools (UnlockTool, Borneo, etc.)  
- 📺 IPTV subscriptions  
- 📞 Customer contact & business info
