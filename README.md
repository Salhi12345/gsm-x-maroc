## 📌 Project Overview

This repository contains static HTML pages used to present:

- Phone unlock and software support services
- IPTV service information
- GSM tool-related pages for customers in Morocco

---

## 📂 Project Structure

| File | Purpose |
|------|---------|
| `index.html` | Main homepage for UnlockTool Maroc |
| `unlocktool.html` | Dedicated page for UnlockTool services |
| `iptv.html` | IPTV-focused service page |
| `borneo.html` | Borneo tool/service information page |
| `*.png`, `*.jpg` | Visual assets used by the pages |

## 👀 How to View the Project

### Option 1 — Live version (recommended)

Open in browser:

**https://gsm-x-morocco-two.vercel.app/**

### Option 2 — Run locally

1. Clone this repository.
2. In the project folder, run:

```bash
python -m http.server 8000
```

3. Open: `http://localhost:8000`
